About
WebRAT is a web based tool that converts relational algebra queries to SQL and optionally queries a MySQL database.

Instructions
  • Press a relational operator button to add it to the relational algebra editor. Each expression is surrounded by parentheses. Click an expression's parentheses to select it. A selected expression can be replaced by clicking a relational operator button.
  • Pressing the "To SQL" button converts the relational algebra in the editor to SQL. The output will appear in the SQL editor.
  • To run a SQL query on a database, press the "Connect" button and fill out the form. Once connected, select the query in the SQL editor that you want to run, then press "Run SQL". The output will be displayed in the box below the SQL editor.
